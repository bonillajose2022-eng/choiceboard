# Kids Choice Board

This project is a colorful and interactive choice board for kids, ready to be published on GitHub Pages.

## Files
- `index.html`
- `styles.css`
- `script.js`

## Features
- Colorful design for children
- 4 clickable cards
- Responsive layout
- Hover effects
- Friendly typography
- Ready for GitHub Pages

## How to use
1. Upload all files to a GitHub repository.
2. Go to repository settings.
3. Enable GitHub Pages from the main branch.
4. Open the published link.

## Edit links
If you want to change the links, open `index.html` and replace the `href` value in each card.

## Notes
You can also change:
- titles
- descriptions
- emojis
- colors
